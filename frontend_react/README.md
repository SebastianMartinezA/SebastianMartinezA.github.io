# Portfolio Frontend

App React 18 + Vite + SCSS para [smar.ar](https://smar.ar/).

## Scripts

Requiere Node.js 22 o superior.

```bash
npm ci
npm run dev
npm test
npm run build
npm run preview
```

## Archivos principales

- `src/App.jsx`: layout, secciones, tema, idioma y acciones principales.
- `src/App.scss`: estilos globales, responsive, tema y micro-interacciones.
- `src/content/portfolioContent.js`: contenido ES/EN, URLs y metadata.
- `src/assets/ResumeSebastian.pdf`: CV publicado.
